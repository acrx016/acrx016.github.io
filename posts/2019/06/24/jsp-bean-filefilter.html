# Bean的辅助类

# Bean

使用Bean列出JSP页面所在目录中特定拓展名的文件 定义一个内部辅助类实现文件过滤








































 
 
 
 
 
 
 
 
 
 
 
 
 

package test.bean;

import java.io.File;
import java.io.FilenameFilter;

public class ListFile {
	String extendsName = null;
	String [] allFileName=null;
	String dir = null;

	public String getExtendsName() {
		return extendsName;
	}


	public void setExtendsName(String extendsName) {
		this.extendsName = extendsName;
	}


	public String getDir() {
		return dir;
	}


	public void setDir(String dir) {
		this.dir = dir;
	}

	public String[] getAllFileName() {
		if(dir!=null) {
			File mulu=new File(dir);
			FileExtendName help=new FileExtendName(extendsName);
			allFileName=mulu.list(help);
		}
		return allFileName;
	}
}

class FileExtendName implements FilenameFilter {
	String str = null;

	FileExtendName(String s) {
		str = "." + s;
	}

	@Override
	public boolean accept(File dir, String name) {
		// TODO Auto-generated method stub
		return name.endsWith(str);
	}
}

# JSP展示页面

JSP页面获取Bean中的值并展示

<%@ page contentType="text/html; charset=UTF-8"%>
<jsp:useBean id="file" class="test.bean.ListFile" scope="request"></jsp:useBean>
<html>
<body>
	<form action="" method="post">
    输入文件目录：<input type="text" name="dir" size=16><br>
    输入拓展名：<input type="text" name="extendsName" size=16><br>
    <input type="submit" value="submit">
	</form>
	<jsp:setProperty property="*" name="file" />
	<br>目录<jsp:getProperty property="dir" name="file" />中
	<br> 拓展名<jsp:getProperty property="extendsName" name="file" />有
	<br>
	<%
		String[] allFileName = file.getAllFileName();
		if (allFileName != null) {
			for (String fileName : allFileName) {
				out.println(fileName+"<br>");
			}
		}
	%>
</body>
</html>

# 辅助类分析

File.list(FilenameFilter filter)方法源码

public String[] list(FilenameFilter filter) {
    String names[] = list();
    if ((names == null) || (filter == null)) {
        return names;
    }
    List<String> v = new ArrayList<>();
    for (int i = 0 ; i < names.length ; i++) {
        if (filter.accept(this, names[i])) {
            v.add(names[i]);
        }
    }
    return v.toArray(new String[v.size()]);
}

TIP

  • 上文list(FilenameFilter filter)方法通过FileFilter接口类对象；
  • 调用其内部方法accept(File pathname)把筛选文件名加入到队列v中；
  • 随后把队列v保存到Bean中；
  • 在JSP页面下引用Bean，使用队列遍历的方式将Bean中队列逐个输出显示；
  • 重写FileFilter接口对应的内部方法：boolean accept(File pathname)
