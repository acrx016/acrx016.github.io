# 第1章 JSP语法

一个JSP页面由5种元素组合而成：

  1. HTML标记符
  2. JSP标记，分为指令标记、动作标记
  3. 变量和方法的声明
  4. Java程序片
  5. Java表达式

# 1.1 常见的JSP指令标记

JSP指令标记对整个页面有效，与书写位置无关，习惯上写在最前面。

# 1.1.1 page指令标记

<%@ page 属性1="属性值" 属性2="属性值" ···%>
1

TIP

page指令标记可以制定如下属性的值：
contentType、import、language、session、buffer、autoFlush、inThreadSafe

WARNING

page指令只能为contentType属性指定一个值

常见用法如下：

/*contentType属性,启用HTML解析器、字符编码UTF-8*/
<%@ page contentType="text/html;charset=UTF-8" %>

/*language属性,只有默认值java，不设置该属性则保持默认*/
<%@ page language="java" %>

/*import属性，引入运行环境包中的类，JSP页面默认已有如下*/
<%@ page import="java.lang.*","javax.servlet.*","javax.sevlet.jsp.*","javax.servlet.http.*" %>

/*session属性用于设置是否需要使用内置session对象，默认true*/
<%@ page session="true" %>

/*buffer属性，设置输出流对象out的缓冲区大小，"none"为不使用缓冲区*/
<%@ page buffer="8KB" %>

/*
    autoFlush属性，配合buffer属性使用，指定out缓冲区被填满时是否自动刷新；
    取值为frue或false，如果为false，buffer被填满就会缓存溢出；
    当buffer值为"none"，则autoFlush属性不能设置成false。
*/

/*isThreadSafe属性,用来设置JSP页面是否可以多线程访问,默认值true*/

/*info属性,其属性值是字符串,可供程序片获取*/
<%@ page info="we are student"%>
//程序片使用如下方法获取字符串值
getServletInfo();

# 1.2 常见的JSP动作标记

JSP动作标记影响JSP页面运行时的功能

# 1.2.1 include动作标记

include动作标记的语法格式为：

<jsp:include page="文件的URL" />
//或者
<jsp:include page="文件的URL">
    param子标记
</jsp:include>
1
2
3
4
5

TIP

与上文include指令标记区别如下：
1.include指令标记在编译阶段处理文件，将子文件与主页面合并，页面执行速度快;
2.include动作标记是在JSP页面运行时处理文件，将处理结果发送到用户端，优点在于使用param子具有灵活性，缺点是执行速度较慢。

# 1.2.2 param 动作标记

param子标记为其他标记提供附加值，不能独立使用，需作为jsp:include、jsp:forward、jsp:plugin标记的子标记使用。 代码用例如下：

/*本例用作向JSP页面传递三角形三边的值*/
<% double a=3,b=4,c=5; %>
加载triangle.jsp计算三边为<%=a %>,<%=b %>,<%=c %>的三角形面积.
<jsp:include page="triangle.jsp">
    <jsp:param name="sideA" value="<%=a%>"/>
    <jsp:param name="sideB" value="<%=b%>"/>
    <jsp:param name="sideC" value="<%=c%>"/>
</jsp:include>

# 第3章 JSP与JavaBean

# 3.1 编写JavaBean

在JSP主要使用非可视Bean，主要设计其属性和方法

  1. 获取属性值的方法getXxx();
  2. 设置属性值的方法setXxx();
  3. 类中声明方法的访问属性是public的();
  4. 类的构造方法可以带参;

# 3.2 JSP动作标记操作Bean

# 3.2.1 jsp:useBean动作标记

<jsp:useBean id="Bean的名字" class="bean的类" scope="bean的生命周期" />
1

TIP

scope生命周期

  • page：当前页面有效；
  • request：页面响应结束（页面执行完毕后）有效，比page略长;
  • session：同一用户会话时间内，同一web服务目录下所有页面共享同一个bean；
  • application：不同用户，同一web服务目录下所有页面共享同一个bean。

# 3.2.2 jsp:getPropety动作标记

<jsp:getPropety name="bean的ID" property="bean的属性" />
/* 相当于 */
<%=bean.getXxx()%>
1
2
3

# 3.2.3 jsp:setPropety动作标记

<jsp:setPropety name="bean的ID" property="bean的属性" value="属性值" />
/* 或者 */
<jsp:setPropety name="bean的ID" property="bean的属性" param="参数" />
1
2
3

WARNING

value属性和param属性均是可选的，但不能同时使用，只能使用一个
param指定用哪个请求参数作为Bean属性的值。
如果当前请求没有参数，则什么事情也不做，系统不会把null传递给Bean属性的set方法。
你可以让Bean自己提供默认属性值，只有当请求参数明确指定了新值时才修改默认属性值。

# 3.4 Bean的计数器

public class Count {
	int number = 0;
	boolean isCome = false;
	File file = new File("count.dat");

	private void countP() {
		if (!file.exists()) {
			number++;
			try {
				file.createNewFile();
				FileOutputStream fileOut = new FileOutputStream(file);
				DataOutputStream dataOut = new DataOutputStream(fileOut);
				dataOut.writeInt(number);
				dataOut.close();
				fileOut.close();
			} catch (Exception e) {
				// TODO: handle exception
			}
		} else {
			// 先把数据读进来，增加1，再写回去
			try {
				FileInputStream fileIn = new FileInputStream(file);
				DataInputStream dataIn = new DataInputStream(fileIn);
				number = dataIn.readInt();
				number++;
				dataIn.close();
				fileIn.close();

				FileOutputStream fileOut = new FileOutputStream(file);
				DataOutputStream dataOut = new DataOutputStream(fileOut);
				dataOut.writeInt(number);
				dataOut.close();
				fileOut.close();
			} catch (Exception e) {
				// TODO: handle exception
			}
		}
		isCome = true;
	}
	public int getNumber() {
		if(isCome==false)
			countP();
		return number;
	}

}

# 4.2 共享变量

TIP

servlet中共享变量的作用效果类似于在JSP页面声明变量时所使用的标记符号<%! %>

public class Computer extends HttpServlet {
	private static final long serialVersionUID = 1L;
	/**
	*	这个部分的变量声明相当于在在JSP页面中使用标记符号<%! %>
	*	以下成员变量被所有访问该servlet的用户共享
	**/
	double p=0;
	double i=1,j=1;
	double count=0;
	public void init(ServletConfig config) throws ServletException {
		super.init(config);
	}
	protected void service(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
		response.setContentType("text/html;charset=utf-8");
		PrintWriter out=response.getWriter();
		out.print("<html><body>");
		p+=i/j;
		i=-i;
		j=j+2;
		count++;
		out.print("</body></html>");
		out.print("servlet"+getServletName()+"被请求"+count+"次");
		out.print("<br> 当前π的值是"+p*4);
	}

}

//servlet
public class Decompose extends HttpServlet {

	private static final long serialVersionUID = 1L;

	public void doPost(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
		request.setCharacterEncoding("utf-8");
		String str=request.getParameter("word");
		if ( str == null)
			str="";
		String []words=str.split("[^a-zA-z]+");

		response.setContentType("text/html;charset=utf-8");
		PrintWriter out = response.getWriter();
		out.println("<HTML>");
		out.println("  <BODY>");
		out.print("句子中所含单词：");
		for(String a:words) {
			out.print(a+" ");
		}
		out.println("  </BODY>");
		out.println("</HTML>");
		out.flush();
		out.close();
	}
}
